## 🚀 How to Share / Deploy

### Option 1: GitHub Pages (Free, Instant)
1. Create a new GitHub repo: `talkto`
2. Push `index.html` to it
3. Go to **Settings → Pages → Deploy from main branch**
4. Your app is live at: `https://BavyaBalakrishnan.github.io/talkto`

### Option 2: Netlify Drop (Fastest — 30 seconds)
1. Go to **https://app.netlify.com/drop**
2. Drag and drop your `talkto` folder
3. Get an instant live URL like: `https://talkto-xyz123.netlify.app`
4. You can rename it to: `https://talkto.netlify.app`

### Option 3: Vercel
1. Run: `npx vercel --prod` inside the `talkto` folder
2. Live in 10 seconds

**I recommend Netlify Drop — it's the fastest, no account needed.**
